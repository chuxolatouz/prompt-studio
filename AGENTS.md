# AGENTS

## Brand + UI Conventions
- Use prompteero assets from `public/brand/` only.
- Keep logo usage compliant: no stretch/rotation/recolor.
- Prefer `Logo` component at `src/components/layout/logo.tsx`.
- Primary CTAs/links use `--prompteero-blue`.
- Orange is accent-only (`--prompteero-orange`), not dominant background.
- Keep UI didactic and simple: clear steps, empty states, short copy.
